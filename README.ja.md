# Barometer - iOS用気圧ロガー

iOS向けのシンプルな気圧および相対高度ロガーです。

## 機能
- 気圧（hPa）と相対高度（m）を記録
- データを日別のCSVファイルに保存
- 現在のCSVファイルを共有可能

## 要件
- iOS 18.5以降
- 気圧センサーを搭載した対応デバイス

## 使い方
1. iOSデバイスにBarometerアプリをインストールします。
2. アプリを起動し、必要な権限を許可します。
3. 「開始」ボタンをタップして、気圧と相対高度の記録を開始します。
4. 記録を停止するには、「停止」ボタンをタップします。
5. 「共有」ボタンをタップして、現在のCSVファイルを共有します。
