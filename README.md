# Barometer - Barometer Logger for iOS

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple barometer and relative altitude logger for iOS.

## Features
- Logs barometric pressure (in hPa) and relative altitude (in meters)
- Saves data to a daily CSV file
- Allows sharing the current CSV file

## Requirements
- iOS 18.5 or later
- Compatible device with a barometric pressure sensor

## Usage
1. Install the Barometer app on your iOS device.
2. Launch the app and grant the necessary permissions.
3. Tap the "Start" button to begin logging barometric pressure and relative altitude.
4. To stop logging, tap the "Stop" button.
5. Tap the "Share" button to share the current CSV file.
